Linux学习之进程管理

发表于 2017-03-20

进程

进程是正在执行的一个程序或命令，每个进程都是一个运行的实体，都有自己的地址空间，并占用一定的系统资源。
主要作用：判断系统是否健康；查看进程；杀死进程。

进程查看命令ps

  • ps aux 查看系统中所有进程，使用BSD操作系统格式
  • ps -le 查看系统中所有进程，使用Linux标准命令格式

进程查看命令pstree

  • -p: 显示进程的PID
  • -u: 显示进程的所属用户

进程查看命令top

  • -d 秒数：指定每隔几秒更新，默认是3秒
  • -b: 使用批处理模式输出（一般与”-n”选项合用）
  • -n 次数：指定命令执行的次数

在top命令的交互模式当中可以执行的命令：

  • ?或h: 显示交互模式的帮助
  • P: 以CPU使用率排序，默认就是此项
  • M: 以内存的使用率排序
  • N: 以PID排序
  • q: 退出top

杀死进程命令kill

kill -l 查看信号

  • kill -1(-HUP) 平滑重启
  • kill -9 强制中止
  • kill -15 默认

杀死进程命令killall

killall [选项][信号]进程名

  • -i: 询问是否要杀死某个进程
  • -I: 忽略进程名的大小写

杀死进程命令pkill

  • -t 终端号：按照终端号踢出用户
    w 查询本机已登录的用户
    pkill -9 -t pts/1 踢出pts/1终端
